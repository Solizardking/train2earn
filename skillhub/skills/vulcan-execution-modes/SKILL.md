# Vulcan Execution Modes

Canonical source: `../../vulcan-cli-master/skills/vulcan-execution-modes/SKILL.md`

Default user-facing mode is Paper unless the user explicitly selects a live mode. Live `confirm_each` and `auto_execute` require explicit approval and collected safety parameters.
