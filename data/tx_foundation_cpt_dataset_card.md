**19,542 Solana transactions** in NeMo CPT (continued pre-training) format. Tokenized by `SolanaTokenizerPipeline` from Jupiter swap quotes and on-chain tx records. Used to train [`solanaclawd/solana-tx-foundation-1.5b`](https://huggingface.co/solanaclawd/solana-tx-foundation-1.5b) (Blueprint 1 of the NVIDIA AI Blueprints Solana integration).

---

## What it contains

Each example is a `{"text": "..."}` record encoding one Solana transaction as a structured token sequence:

```text
PROG_89 IX_SWAP MINT_2840 MINT_3620 AMT_73 AMT_66 FEE_2 SLOT_110 SIDE_BUY STATUS_SUCCESS
```

Field types:

| Field | Token prefix | Description |
| --- | --- | --- |
| Program | `PROG_N` | Involved program IDs (bucketed by frequency) |
| Instruction | `IX_TYPE` | Instruction discriminator / name |
| Mint | `MINT_N` | Token mint addresses (bucketed) |
| Amount | `AMT_N` | Transfer amounts (log-bucketed) |
| Fee | `FEE_N` | Transaction fee in lamports (log-bucketed) |
| Slot | `SLOT_N` | Relative slot (bucketed) |
| Side | `SIDE_BUY` / `SIDE_SELL` | Trade direction for swaps |
| Status | `STATUS_SUCCESS` / `STATUS_FAIL` | Transaction outcome |

Vocabulary size: **4,886** tokens.

Some records use raw Jupiter swap quote context rather than the tokenized format. Both types are included so the model learns to handle real API output as well as the compact token representation.

## Format

```json
{"text": "PROG_89 IX_SWAP MINT_2840 MINT_3620 AMT_73 AMT_66 FEE_2 SLOT_110 SIDE_BUY STATUS_SUCCESS"}
```

or (Jupiter quote context format):

```json
{"text": "<tx_context>\nJupiter swap quote [2026-06-19T...]\nRoute: SOL → USDC via Whirlpool\n...\n</tx_context>"}
```

## Source

Generated by `nvidia/blueprints/transaction-foundation-model/pipeline.py --stages cpt` from:

- Jupiter Price API v2 swap quotes (SOL, BTC, ETH, JUP, PYTH)
- On-chain Solana transaction records (mainnet-beta)
- Tokenizer: `nvidia/blueprints/transaction-foundation-model/src/tokenizer/solana_tokenizer.py`

## Usage

### Load as a HuggingFace dataset

```python
from datasets import load_dataset

ds = load_dataset("solanaclawd/solana-tx-foundation-cpt")
print(ds["train"][0])
# {"text": "PROG_89 IX_SWAP MINT_2840 ..."}
```

### Use for CPT with TRL SFTTrainer

```python
from datasets import load_dataset
from trl import SFTConfig, SFTTrainer
from transformers import AutoModelForCausalLM, AutoTokenizer

model_id = "Qwen/Qwen2.5-1.5B-Instruct"
tokenizer = AutoTokenizer.from_pretrained(model_id)
model = AutoModelForCausalLM.from_pretrained(model_id, torch_dtype="auto", device_map="auto")

ds = load_dataset("solanaclawd/solana-tx-foundation-cpt", split="train")

trainer = SFTTrainer(
    model=model,
    args=SFTConfig(
        output_dir="./outputs/solana-tx-foundation-cpt",
        max_length=512,
        num_train_epochs=1,
        per_device_train_batch_size=4,
        gradient_accumulation_steps=4,
        learning_rate=5e-5,
        dataset_text_field="text",
    ),
    train_dataset=ds,
    processing_class=tokenizer,
)
trainer.train()
```

### Regenerate from the pipeline

```bash
cd ai-training
python3 nvidia/blueprints/transaction-foundation-model/pipeline.py --stages cpt
```

## Intended Use

- Continued pre-training of small language models (1.5B–7B) on Solana transaction semantics
- Foundation for fine-tuning on-chain analysis, anomaly detection, and DeFi routing models
- Blueprint 1 of the [NVIDIA AI Blueprints](https://build.nvidia.com/nvidia/build-your-own-transaction-foundation-model) Solana integration

## Out of Scope

- Live wallet balances, private keys, or personally identifiable information
- Real trading signals or guaranteed financial outcomes
- Front-running or MEV exploitation
